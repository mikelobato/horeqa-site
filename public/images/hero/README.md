# Hero Assets

Drop generated hero images here using these names:

- `hero_horeqa_01_overlay.webp` (desktop, preferred)
- `hero_horeqa_01_overlay.jpg` (desktop fallback)
- `hero_horeqa_01_mobile.webp` (mobile, preferred)
- `hero_horeqa_01_mobile.jpg` (mobile fallback)

Current `components/hero.tsx` reads these files automatically.

Recommended export sizes:

- Desktop: `1920x720` or `1920x1080`
- Mobile: `1080x1350`

Keep images free of embedded text/logos for overlay readability.
